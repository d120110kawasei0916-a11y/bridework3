# BridgeWork（静的版）

単一ページの求人管理ツールです。GitHub にドラッグ＆ドロップで公開できます。

## ファイル構成（2ファイル）
- `index.html` … アプリ本体（React/Tailwind は CDN）
- `README.md` … 使い方

## 使い方（GitHub Pages）
1. GitHub で新しいリポジトリを作成（例: `bridgework`）。
2. この ZIP を展開して **中の2ファイル**をそのままリポジトリのルートにドラッグ＆ドロップでアップロード。
3. **Settings → Pages** → Source を `Deploy from a branch`、Branch を `main`、Folder を `/ (root)` に設定して保存。
4. 数十秒で公開URLが発行されます。
   - 起動後、「掲載 or 閲覧」を選択（閲覧のパスワードは **1416**）。
   - 「求人所有会社名」はプレースホルダーに **例：〇〇紹介会社**。
   - 「説明」は **任意**。
   - **職種**を入れると **職種ごとの連番**（例: `介護職-001`）が自動採番されます。

## 注意点
- データはブラウザの `localStorage` に保存（PCごとに別データ）。
- 添付ファイルはURL推奨。画像やPDFを大量に保存すると容量上限（約5〜10MB/ブラウザ）に到達します。
- 共有・同時編集が必要になったら Firebase/Supabase 版への拡張が可能です。
